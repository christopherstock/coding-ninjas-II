# Primary npm Targets

## Install all Node.js packages
```
npm install --legacy-peer-deps
```

## Start the Webpack Dev Server and build the project
```
npm webpack:devserver
```

## Run the app
Visit the following URL in your browser:
```
http://localhost:1234
```
